Fuzzy Cognitive Map Theory Guide

What is a Fuzzy Cognitive Map (FCM)?

Introduced by Bart Kosko in 1986, Fuzzy Cognitive Maps are soft-computing causal models used to describe, simulate, and analyze complex systems of interconnected factors. They function like an artificial neural network with customizable activation pathways.

Key Concepts

  • Concept Nodes ($A_i$): System elements (e.g. "Public Transit Quality", "Stress Level"). Each node has an activation value between 0.0 and 1.0 (or -1.0 and +1.0 in Bipolar models) representing its magnitude or level of occurrence.
  • Directed Edges ($W_{ji}$): Direct causal connections showing how concept $j$ influences concept $i$. The weight $W_{ji}$ falls in the range [-1.0, 1.0]:
    • $W_{ji} > 0$ (Positive Causality): An increase in Concept $j$ causes an increase in Concept $i$.
    • $W_{ji} < 0$ (Negative Causality): An increase in Concept $j$ causes a decrease in Concept $i$.

Simulation Math

During a simulation, concept values update over discrete iterations using either:

  1. Standard Kosko: $A_i(t+1) = f(\sum A_j(t) \cdot W_{ji})$
  2. Modified Kosko: $A_i(t+1) = f(\sum A_j(t) \cdot W_{ji} + A_i(t))$ (Includes the concept's historical memory to avoid fading values)

The transfer function $f(x)$ bounds the outputs. We support:

  • Sigmoid: Compresses values into [0.0, 1.0]. Ideal for strictly positive concepts.
  • Bipolar Sigmoid: Compresses values into [-1.0, 1.0]. Perfect for concepts that can be fully negative (e.g., negative budget).
  • Linear Clamped: Standard linear mapping bounded strictly by system borders.
